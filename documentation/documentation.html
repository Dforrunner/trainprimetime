Installation

This site was build using Next.js . You'll need to have an understanding of how that works to be able to edit this project as you wish and the prerequisite to Next.js is React, which you also need to know. Check the Resources section to learn more about the libraries used in this project and where to learn more about them.

Follow the steps below to setup the site for development

  1. First make sure you have Node and npm installed. These are needed to run the code
    • To check if you have them installed open terminal and simply type the commands highlighted in black and press enter
      • node -v - you should see v16.17.0 or higher
      • npm -v - you should see 8.19.0 or higher
    • If you get an error, don't see the expected result or your version is lower than the ones shown above, then go to the following site and download Node. Note: npm is install along with Node so you don't need to worry about that once you install Node.
      • Node download - Download Node for your operation system
      • Run node -v to confirm it's downloaded - you should see v16.17.0 or higher
  2. Open terminal or command line and navigate to the project root directory
    • Run npm i to install all the dependencies
    • Once that's done run npm run dev to start the development server
    • Open http://localhost:3000 with your browser to see the result

Database

Documentation for database used, access and manipulation

The database used in this app is PostgresSQL. You can read more about it on their website. However, it's not really necessary since an ORM is used to connect and manipulate data in the database. The ORM used in this project is Prisma.

  • Prisma offers a nice user interface to view and edit database in your browser called the Prisma Studio.
    • To open Prisma Studio run the following commend in the terminal. Note: You'll need to be in the project directory to run these commands.
      • prima studio or npx prima studio
      • Go to http://localhost:5555 to view the Prisma Studio
  • Prisma offers database connections to varies different database options. Read more about that here.
    • To connect to a different database change the DATABASE_URL variable in the /.env file located in the project root directory.
  • Database schema is located in at /prima/schema.prisma
